Este proyecto es un ejercicio obligatorio del modulo 5 del curso de MiriadaX. 
Donde hay que clonar el proyecto "random" del proyecto https://github.com/jquemada/random.
Y añadir un tercer commit con los ficheros readme.md y license.md.
 
Ese proyecto clonado tiene un archivo random.js, el cual imprime por consola un número aleatorio entre 0 y 1, y dice si ese número es menor o mayor que 0,5.
